# gsi-cyber-japan-old
IITCのベースマップレイヤーに、国土地理院より公開されている昔の航空写真を追加します。<br />

# 使い方
IITCの導入後、<a href="https://github.com/mklyr/gsi-cyber-japan-old/raw/main/basemap-gsi-old-photo.user.js">ここからダウンロードして下さい。</a><br />
あとは画面に従ってインストールするだけです。<br />
